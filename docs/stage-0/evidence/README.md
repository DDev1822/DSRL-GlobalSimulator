# Evidencias visuales — Etapa 0

Esta carpeta recibirá únicamente capturas obtenidas después de ejecutar:

```bash
npm run verify:baseline
npm run dev
```

## Archivos requeridos

1. `01-dashboard-fullscreen.png` — dashboard completo en pantalla completa.
2. `02-curve-hover.png` — línea y tres marcadores móviles activos sobre la curva.
3. `03-pit-phase-f1.png` — pit en F1.
4. `04-pit-phase-f6.png` — pit en F6.
5. `05-economic-layer.png` — una capa económica distinta de componente/elevación.
6. `06-wireframe.png` — pit con wireframe activo.

## Reglas

- Zoom del navegador: 100 %.
- Sin errores visibles ni cuadros negros.
- La lectura del cursor debe permanecer fuera del área principal del pit.
- No capturar información privada del navegador o del sistema.
- No sustituir una evidencia requerida por una captura de otra función.

La Etapa 0 no se considera cerrada hasta completar las seis evidencias y la lista `manual-visual-checklist.md`.
